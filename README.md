## Install

```bash
npm i csvs-parsers@latest
```

With `yarn`

```bash
yarn add csvs-parsers@latest
```

## Use

```js
import { csvToJson } from 'csvs-parsers';


const start = async () => {
 const results = await csvToJson({
    pathUrl: url // path name or URL vsv,
  });
}

start();
```
